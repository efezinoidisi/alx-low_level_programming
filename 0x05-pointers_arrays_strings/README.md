## Pointers, Arrays and Strings

This directory contains C programs related to arrays, pointers and strings

- 0-reset_to_98.c is a C function that takes a pointer to an int and updates the value it points to, to 98.

- 1-swap.c is a function that swaps the value of two integers.

- 2-strlen.c is function that returns the length of a string.

- 3-puts.c is a function that prints a string followed by a new line to standard output.

- 4-print_rev.c is a function that prints a string in reverse followed by a new line.
